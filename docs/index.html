Cryptographic Proof

AI either hallucinates facts or gets them from tools.
TrustChain signs every fact with Ed25519 — and hands you a portable .tcreceipt that anyone can verify offline.

View on GitHub Quick Start
sign_and_export.py
from trustchain import TrustChain, build_receipt

# Sign every fact your AI pulls from a real tool (Ed25519 + auto-chain)
tc = TrustChain()
signed = tc.sign(
    tool_id="database_query",
    data={"user_id": 42, "balance": 1234.56},
)
assert tc.verify(signed) # True

# Export a portable proof — JSON file anyone can verify offline
receipt = build_receipt(signed, tc.export_public_key())
open("balance.tcreceipt", "w").write(receipt.to_json())
Why TrustChain

Ed25519 Signatures

New · v3.0.0

The .tcreceipt Format

A single JSON file = portable proof that a signed fact is real. Verify it anywhere — browser, CLI, Python, JS, airgapped laptop — without ever contacting our servers.

Download anything

One click on any signed AI output — chat answer, tool result, RAG citation — exports a .tcreceipt JSON with envelope + public key + optional identity.

Verify offline

Drag & drop into a single HTML file — WebCrypto Ed25519 check, no network, no install. Works airgapped.

Byte-exact parity

Python (tc receipt verify), JS SDK, Browser WebCrypto — all three produce bit-identical canonical envelopes. Covered by automated parity tests.

Tamper-evident & pinned

Exit codes (0/2/3/4) distinguish ok / tampered / degraded / malformed. Optional pinning by key_id, max age, and issuer CN.
