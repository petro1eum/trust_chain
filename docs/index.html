Cryptographic Proof
for AI Agent Actions

AI either hallucinates facts or gets them from tools.
TrustChain signs every fact from real tools. Signature = trust.

View on GitHub Quick Start
from trustchain import TrustChain

# Sign AI tool responses with Ed25519
tc = TrustChain()
response = tc.sign_response(
    tool_id="database_query",
    output="User balance: $1,234.56"
)

# Verify response integrity
assert tc.verify(response) # True

Enterprise-Grade Security

Everything you need to build trustworthy, auditable AI systems

Ed25519 Signatures

Every tool call is cryptographically signed with fast, secure Ed25519 elliptic curve signatures.

Chain of Trust

Link responses together with parent hashes. Prove the complete execution history of any AI action.

Replay Protection

Built-in nonce and timestamp validation prevents replay attacks and ensures freshness.

Policy Engine

Define YAML-based policies for runtime enforcement. Allow, deny, or require specific tool patterns.

Execution Graphs

Visualize agent execution as DAGs. Detect forks, replays, and orphaned responses automatically.

Merkle Audit Trail

Append-only audit logs with Merkle tree verification. Perfect for compliance and forensics.
